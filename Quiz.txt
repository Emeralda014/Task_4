
A cookie is a small file that a server stores only on the client or user computer.
The purpose of cookies is remembering information about the user, and the user's site preferences.
A cookie can only be visible to the user that created it and the data generated are only stored on the client-side machine,
and they can be disabled by the user in most web browsers.
They expire depending on the lifetime a user sets for it.
